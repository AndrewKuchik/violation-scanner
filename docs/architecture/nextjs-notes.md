# ⚠️ Next.js 16 — версия НЕ как в обучении модели

Эта версия Next.js имеет ломающие изменения. API, соглашения и структура файлов могут отличаться от того, что модель «помнит».

**Правило:** перед написанием любого кода Next.js читай нужный гайд в
`node_modules/next/dist/docs/` и соблюдай пометки об устаревании (deprecation).

Точная версия: `next@16.2.10`, `react@19.2.4` (см. `package.json`).

Специфика проекта:
- `/api/scan` — Node.js runtime (не Edge): Playwright требует полноценный Node.
- Приложение запускается как долгоживущий процесс (`next start`), а не serverless — скан на 10-30 сек не упирается в лимит запроса.
