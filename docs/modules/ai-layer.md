# Модуль: AI-объяснения (Ф.5, опционально)

⚠️ Работа с моделью Claude — при задачах по этому модулю сверься с актуальными данными API (модель/параметры), не по памяти.

## Суть
Опциональный слой обогащения. По умолчанию **выключен** (режим «Только правила» работает без ключа). Включается `AI_EXPLANATIONS_ENABLED=true` + `ANTHROPIC_API_KEY`.

## Как устроено
- **Один батч-запрос** Claude на весь скан (не по вызову на находку — дешевле, быстрее, кэшируется общий системный промпт).
- Модель: `claude-opus-4-8`.
- Структурированный вывод: `output_config: {format: {type:"json_schema", schema: ...}}` — валидируется и мапится обратно на `Finding[]`. НЕ через forced tool_use.
- Модели передаются готовые `legalRefs` (статично из правила). Её задача — только перефразировать понятным языком + дать шаги по исправлению.
- Системный промпт **явно запрещает**: придумывать номера статей; любые формулировки о гарантированном соответствии закону (см. `product/legal-guardrails.md`).

## Файлы
- `lib/ai/explain.ts` — батч-вызов.
- `lib/ai/prompts.ts` — системный промпт + JSON-схема.
