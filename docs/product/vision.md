# Видение продукта

**Elevator pitch:** пользователь вставляет ссылку на любой сайт → за 10–30 сек сервис открывает его в реальном браузере, ловит что происходит на самом деле (куки/трекеры до согласия, баннер, политика, HTTPS), сопоставляет с базой «нарушение → статья закона → диапазон штрафа» и выдаёт понятный отчёт: **что не так, где именно, во сколько обойдётся**.

**Ниша (подтверждена ресёрчем):** никто не соединяет три элемента сразу — (что / где / сколько). Cookiebot/CookieYes/Osano сканируют, но не оценивают штраф. FineMyData оценивает по анкете, но не по скану и не показывает «где». Мы — единственные.

**Аудитория v1:** любой сайт. Приоритетный сегмент для демо/маркетинга — **интернет-магазины** (максимум нарушений, богаче находки, выше мотивация владельца платить). Сканер при этом не ограничен только e-commerce.

## Roadmap после MVP (не обязательство, материал для обсуждения)
PDF-экспорт · мониторинг во времени · многостраничный обход · сравнение конкурентов · API для агентств · white-label дашборд · многоязычный отчёт (LV/EN/RU) · персонализация оценки по обороту · публичный «значок скана» (⚠️ юр-риск) · правила под другие юрисдикции ЕС · browser-расширение.

## Монетизация (отложена до валидации идеи)
Бесплатный разовый скан → платный PDF/мониторинг по подписке; тариф для агентств; pay-per-scan API. Сейчас фокус — чтобы сканер работал и был полезен.
