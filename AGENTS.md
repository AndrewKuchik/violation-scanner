# Как читать этот проект (экономия контекста)

НЕ читай всю документацию подряд. Сначала открой **`docs/INDEX.md`** — это карта: найди свою задачу и прочитай только указанные там 1–3 файла. Статус работ и «что делать дальше» — в `docs/roadmap/phases.md`. Закончил задачу — обнови там статус.

Полные архивные документы (`Planes/стратегия.md`, `Planes/технический-план-исходный.md`) читай целиком только если явно нужна вся картина.
